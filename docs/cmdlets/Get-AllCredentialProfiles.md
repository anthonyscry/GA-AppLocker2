# Get-AllCredentialProfiles

## SYNOPSIS
Retrieves all credential profiles.

## SYNTAX

```
Get-AllCredentialProfiles [<CommonParameters>]
```

## DESCRIPTION
Gets all saved credential profiles with summary information.

## EXAMPLES

### EXAMPLE 1
```
Get-AllCredentialProfiles
```
