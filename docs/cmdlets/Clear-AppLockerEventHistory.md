# Clear-AppLockerEventHistory

## SYNOPSIS
Clears event history.

## SYNTAX

```
Clear-AppLockerEventHistory [<CommonParameters>]
```

## DESCRIPTION
Removes all entries from the event history buffer.

## EXAMPLES

### EXAMPLE 1
```
Clear-AppLockerEventHistory
```
