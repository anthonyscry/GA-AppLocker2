# Clear-SessionState

## SYNOPSIS
Clears the saved session state file.

## SYNTAX

```
Clear-SessionState [<CommonParameters>]
```

## DESCRIPTION
Removes the session.json file to start with a clean slate.

## EXAMPLES

### EXAMPLE 1
```
Clear-SessionState
```
