# Get-AuditLogSummary

## SYNOPSIS
Gets a summary of audit log activity.

## SYNTAX

```
Get-AuditLogSummary [[-Days] <Int32>] [<CommonParameters>]
```

## DESCRIPTION
Gets a summary of audit log activity.
Returns the requested data in a standard result object.

## EXAMPLES

### EXAMPLE 1
```
Get-AuditLogSummary -Days 7
```
