# Stop-RuleIndexWatcher

## SYNOPSIS
Stops monitoring the Rules directory.

## SYNTAX

```
Stop-RuleIndexWatcher [<CommonParameters>]
```

## DESCRIPTION
Stops monitoring the Rules directory.
Gracefully stops the running operation.
