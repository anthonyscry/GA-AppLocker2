# Get-AppLockerStandardEvents

## SYNOPSIS
Gets list of standard event names.

## SYNTAX

```
Get-AppLockerStandardEvents [<CommonParameters>]
```

## DESCRIPTION
Returns the list of standard events used by GA-AppLocker.

## EXAMPLES

### EXAMPLE 1
```
Get-AppLockerStandardEvents
```
